# bird-multimedia-management-system

This project is a multimedia management application for storing, processing, and recognizing bird species using images and audio recordings.
It was developed using Oracle SQL Developer for the database and C# (WebForms in Visual Studio) for the frontend interface.

The system is designed for educational, research, and ornithology enthusiasts, enabling users to manage multimedia resources and perform semantic image recognition based on visual similarity.

Features

-Image Management: Store and retrieve bird images; Resize, rotate, and adjust brightness; Generate and compare image signatures using ORDImageSignature for similarity detection

-Audio Management: Store and retrieve bird audio recordings; Manage file type, duration, and metadata

-Semantic Recognition: Identify similar bird species based on color, texture, shape, and spatial location

Web Interface (C# / WebForms): Upload and display multimedia content; Query the database through an intuitive interface

Technologies Used

Oracle Database with ORDSYS.ORDIMAGE and ORDSYS.ORDIMAGESIGNATURE
SQL Developer (PL/SQL procedures)
C# (ASP.NET WebForms) in Visual Studio
